# 分享大纲

## 前文

CSS 作为前端三架马车之一，有着不可或缺的地位，也是 WEB 开发中必不可少的一环。但在前端工程化日益演进的今天，针对 CSS 的改进却很少，因为 CSS 有着太多兼容性问题，且大都无法进行 polyfill 处理，毫不客气的说目前能使用的技术大多属于十年前的产物了。

今天在这里我将带你了解前端 CSS 的发展进程和现代化 CSS 的使用，让你能提前用上下一个十年的新特性~
